# VerumForma Image Assets

## Folder structure

- `projects/` — Project photography. Name files by project slug, e.g. `avenida-norte.jpg`, `cascais-villa.jpg`.
- `team/` — Team member portraits. Name files by person, e.g. `joao-silva.jpg`.
- `general/` — General site imagery, backgrounds, and hero visuals.

## Naming conventions

- Use lowercase, hyphens only (no spaces or underscores).
- Include a short descriptor: `project-slug-01.jpg`, `project-slug-exterior.jpg`.
- Preferred format: `.jpg` for photos (quality 85), `.png` for images requiring transparency, `.svg` for icons and logos.
